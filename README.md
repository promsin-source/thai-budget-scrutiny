# ทีมตรวจสอบงบประมาณ (Thai Budget Scrutiny)

ปลั๊กอินสำหรับ **ชั้นกรรมาธิการพิจารณาร่างพระราชบัญญัติงบประมาณรายจ่ายประจำปี** ช่วยตรวจสอบคำของบประมาณของหน่วยรับงบประมาณอย่างเป็นระบบ แล้วสรุปเป็น **บันทึกคำถามกรรมาธิการ** พร้อมใช้ในการประชุม

ออกแบบให้ทำงานกับชุดเอกสารงบประมาณมาตรฐานของสำนักงบประมาณ (เอกสารงบประมาณ ฉบับที่ 1–7, ร่าง พ.ร.บ., งบประมาณโดยสังเขป, บันทึกวิเคราะห์ฯ) ใช้ได้ทุกปีงบประมาณ (ปัจจุบันตั้งค่ากรอบตัวเลขอ้างอิงไว้ที่ปีงบประมาณ พ.ศ. 2570)

## ทีมเอเจนต์ (Agents)

| เอเจนต์ | บทบาท |
|---|---|
| `budget-lead` | หัวหน้าทีม — รับหน่วยงานเป้าหมาย มอบงานให้ผู้เชี่ยวชาญ 4 ด้าน รวบรวมข้อค้นพบ และร่างบันทึกคำถามกรรมาธิการ |
| `fiscal-discipline-analyst` | ตรวจวินัยการเงินการคลัง/ภาพรวม — กรอบวงเงิน สัดส่วนรายจ่ายลงทุน งบผูกพัน ความสอดคล้อง พ.ร.บ.วินัยการคลังฯ |
| `duplication-checker` | ตรวจความซ้ำซ้อน/ทับซ้อน — โครงการคล้ายกันข้ามหน่วยงาน งบบูรณาการ ภารกิจซ้ำ |
| `value-for-money-analyst` | ตรวจความคุ้มค่า/เหตุผลความจำเป็น — ผลผลิต-ผลลัพธ์ ตัวชี้วัด อัตราการเบิกจ่าย เงินกันเหลื่อมปี เงินนอกงบประมาณ |
| `cost-commitment-auditor` | ตรวจครุภัณฑ์/ราคากลาง & งบผูกพัน — เทียบราคามาตรฐาน จำนวนที่ตั้ง โครงการก่อหนี้ผูกพันข้ามปี |

## สกิล (Skills)

| สกิล | ใช้เมื่อ |
|---|---|
| `budget-scrutiny` | "ตรวจงบของ [หน่วยงาน]" — จุดเริ่มต้น สั่งทีมทำงานครบ 4 มุมแล้วสรุป |
| `committee-questions` | "ทำบันทึกคำถามกรรมาธิการ" — แปลงข้อค้นพบเป็นเอกสาร Word ภาษาไทยพร้อมใช้ |

## เอกสารอ้างอิงภายใน (references)

อยู่ใน `skills/budget-scrutiny/references/`
- `scrutiny-lenses.md` — เช็กลิสต์ตรวจสอบละเอียดทั้ง 4 มุม
- `budget-doc-map.md` — แผนที่เอกสารงบประมาณ ฉบับ 1–7 และวิธีค้นหาข้อมูลของหน่วยงาน
- `fiscal-framework.md` — กรอบวงเงิน สัดส่วน ตัวเลขจริงปี 2570 และเกณฑ์กฎหมายที่ใช้ตรวจ
- `committee-question-template.md` — รูปแบบและตัวอย่างบันทึกคำถามกรรมาธิการ

## วิธีเริ่มใช้

1. วางชุดเอกสารงบประมาณ (PDF ของสำนักงบประมาณ) ไว้ในโฟลเดอร์โปรเจกต์
2. พิมพ์ เช่น *"ตรวจงบของกระทรวงศึกษาธิการ"* หรือ *"ตรวจงบผูกพันของกรมทางหลวง"*
3. ทีมจะอ่านเอกสาร ตรวจ 4 มุม และเสนอ **บันทึกคำถามกรรมาธิการ** เป็นไฟล์ Word

## ข้อจำกัดและหลักการ

- เครื่องมือนี้ **ช่วยตั้งคำถามและชี้ประเด็น** ไม่ใช่คำวินิจฉัย ทุกตัวเลข/ข้ออ้างต้องอ้างอิงหน้าเอกสารต้นฉบับ และผู้ใช้ต้องตรวจทานก่อนนำเข้าที่ประชุม
- กรรมาธิการ **ปรับลดได้ ไม่เพิ่ม** และห้ามแปรญัตติเพิ่ม/แทรกให้ตนเอง ตามรัฐธรรมนูญ ม.144
- ระบุแหล่งที่มา (ฉบับ/เล่ม/หน้า) ทุกครั้งที่อ้างตัวเลข

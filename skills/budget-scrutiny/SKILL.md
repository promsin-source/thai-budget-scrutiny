# ตรวจสอบงบประมาณหน่วยรับงบประมาณ (ชั้นกรรมาธิการ)

จัดการการตรวจสอบคำของบประมาณอย่างเป็นระบบ ตั้งแต่รับหน่วยงานเป้าหมาย → ค้นเอกสาร → ตรวจ 4 มุม → สรุปเป็นบันทึกคำถามกรรมาธิการ

## หลักการสำคัญ (ต้องยึดเสมอ)
- **อ้างอิงเอกสารต้นฉบับทุกครั้ง** — ทุกตัวเลข/ข้ออ้างต้องระบุ (เอกสารฉบับที่ __ / เล่ม __ / หน้า __) ห้ามเดาตัวเลข ถ้าไม่มีข้อมูลให้ระบุว่า "ต้องขอเพิ่มจากหน่วยงาน"
- **ตั้งคำถาม ไม่ตัดสิน** — ผลลัพธ์คือ "ประเด็นให้หน่วยงานชี้แจง" ไม่ใช่คำวินิจฉัยว่าผิด/ทุจริต
- **กรรมาธิการปรับลดได้ ไม่เพิ่ม** — และห้ามแปรญัตติในลักษณะที่ต้องห้ามตามรัฐธรรมนูญ ม.144
- **ภาษาไทย เป็นทางการ กระชับ** — เหมาะกับเวทีกรรมาธิการ

## ขั้นตอน
1. **ยืนยันขอบเขต** — ถามสั้น ๆ ถ้ายังไม่ชัด: หน่วยงานใด, ปีงบประมาณใด (ค่าเริ่มต้น 2570 ตามเอกสารในโปรเจกต์), เจาะแผนงาน/โครงการใดเป็นพิเศษหรือตรวจทั้งหน่วยงาน
2. **ค้นเอกสาร** — เปิด `references/budget-doc-map.md` เพื่อรู้ว่าข้อมูลของหน่วยงานอยู่ไฟล์ไหน แล้วดึงหน้าที่เกี่ยวข้องด้วย Read (รองรับ PDF) หรือ Bash + pdfplumber สำหรับการค้นข้ามไฟล์
3. **ตรวจ 4 มุม** — อ่าน `references/scrutiny-lenses.md` แล้วดำเนินการ โดยอาจ:
   - **แนะนำ:** มอบหมายให้เอเจนต์เฉพาะทางผ่าน Task tool — `fiscal-discipline-analyst`, `duplication-checker`, `value-for-money-analyst`, `cost-commitment-auditor` (ส่ง context: ชื่อหน่วยงาน + ไฟล์/หน้าที่ดึงมา + ตัวเลขวงเงิน) เรียกพร้อมกันได้เพื่อความเร็ว
   - หรือถ้าไม่ใช้ subagent: ตรวจเองตามเช็กลิสต์ทั้ง 4 หัวข้อใน `scrutiny-lenses.md`
4. **สังเคราะห์** — รวมข้อค้นพบ ตัดซ้ำ จัดกลุ่มตามแผนงาน/โครงการ ให้ระดับความสำคัญ 🔴 สูง / 🟡 กลาง / 🟢 ต่ำ ตามเกณฑ์: วงเงินสูง, ความเสี่ยงชัด, กระทบประชาชนวงกว้าง, ประเด็นกฎหมาย/วินัยการคลัง
5. **ออกผลลัพธ์** — เรียกสกิล `committee-questions` เพื่อจัดทำ **บันทึกคำถามกรรมาธิการ** เป็นไฟล์ Word ภาษาไทย (ถ้าผู้ใช้อยากได้แค่สรุปบนแชตก่อน ให้สรุปตามรูปแบบด้านล่างแล้วค่อยถามว่าจะออกเป็น Word ไหม)

## รูปแบบสรุปบนแชต (ก่อนออก Word)
1. **สรุปผู้บริหาร** 3–5 บรรทัด: วงเงินรวมของหน่วยงาน, จำนวนประเด็นแยกตามระดับ, 3 ประเด็นเด่น
2. **ตารางประเด็น**: ระดับ | แผนงาน/โครงการ | วงเงิน (ลบ.) | มุมตรวจ | ข้อสังเกต | อ้างอิง(ฉบับ/หน้า)
3. **ร่างคำถาม** เรียงตามลำดับความสำคัญ

# รูปแบบและตัวอย่าง — บันทึกคำถามกรรมาธิการ

โครงสร้างมาตรฐานของ "บันทึกคำถามกรรมาธิการ" (ชุดคำถามซักหน่วยงานในชั้นกรรมาธิการ) พร้อมตัวอย่างการเขียนคำถามที่ดี

## โครงสร้างเอกสาร
```
บันทึกคำถามกรรมาธิการ
การพิจารณาร่าง พ.ร.บ. งบประมาณรายจ่ายประจำปีงบประมาณ พ.ศ. ๒๕๗๐

หน่วยรับงบประมาณ : ......................................................
วงเงินที่เสนอขอ  : ...................... ล้านบาท (เทียบปีก่อน ......%)
ผู้จัดทำ/คณะทำงาน : ..............   วันที่ : ..............

๑. สรุปภาพรวม (Executive Summary)
   - วงเงินรวม/โครงสร้างโดยย่อ, จำนวนประเด็นแยกตามระดับ (สูง/กลาง/ต่ำ), ประเด็นเด่น ๓ ข้อ

๒. ประเด็นคำถามรายข้อ (จัดกลุ่มตามแผนงาน/โครงการ, เรียงตามลำดับความสำคัญ)
   [รูปแบบต่อประเด็น — ดูด้านล่าง]

๓. ตารางสรุปประเด็นและวงเงินที่เสนอให้พิจารณา

๔. เอกสาร/ข้อมูลที่ขอให้หน่วยงานจัดส่งเพิ่มเติม
```

## รูปแบบประเด็นคำถามรายข้อ
> **ประเด็นที่ [ลำดับ]** — [ระดับ 🔴/🟡/🟢] — [มุมตรวจ]
> **แผนงาน/โครงการ/รายการ:** ...........  **วงเงิน:** ........ ลบ. **(อ้างอิง: ฉบับที่ __/เล่ม __/หน้า __)**
> **ข้อสังเกต:** [ข้อเท็จจริง+ตัวเลข ที่ทำให้เกิดข้อสงสัย — อ้างอิงชัด]
> **คำถามต่อหน่วยงาน:**
>   (๑) ...
>   (๒) ...
> **ข้อเสนอเชิงพิจารณา:** [เช่น ขอทบทวน/ขอเอกสารเพิ่ม/เสนอให้พิจารณาปรับลด ...... ลบ. โดยระบุฐาน]

## หลักการเขียนคำถามที่ดี
- **อิงข้อเท็จจริงและตัวเลข** ก่อนเสมอ แล้วจึงตั้งคำถาม — ไม่ใช้ถ้อยคำกล่าวหา/ชี้นำว่าทุจริต
- **เฉพาะเจาะจง วัดได้** — ระบุรายการ วงเงิน หน้าเอกสาร
- **ปลายเปิดให้ชี้แจง** — "ขอทราบเหตุผลความจำเป็น...", "ขอทราบที่มาของราคา...", "ขอทราบผลการดำเนินงานปีที่ผ่านมา..."
- **ผูกกฎหมาย/มาตรฐาน** เมื่อเกี่ยวข้อง (พ.ร.บ.วินัยการเงินการคลังฯ, ม.41 ผูกพัน, ราคามาตรฐานครุภัณฑ์)
- **เคารพกรอบรัฐธรรมนูญ ม.144** — ปรับลดได้ ไม่เพิ่ม/ไม่แทรกเพื่อประโยชน์ตน

## ตัวอย่างคำถามรายมุม (โทนพร้อมใช้)
**มุมวินัยการคลัง:** "ปีงบประมาณ ๒๕๗๐ หน่วยงานตั้งรายจ่ายลงทุนคิดเป็นร้อยละ __ ของวงเงินหน่วยงาน ลดลงจากปีก่อนร้อยละ __ ขอทราบเหตุผล และมาตรการรักษาสัดส่วนการลงทุนตามกรอบ พ.ร.บ.วินัยการเงินการคลังฯ"

**มุมซ้ำซ้อน:** "โครงการพัฒนาแพลตฟอร์มข้อมูล [ก] วงเงิน __ ลบ. มีลักษณะใกล้เคียงกับระบบ [ข] ของ [อีกกรม/แผนบูรณาการรัฐบาลดิจิทัล] ขอทราบการแบ่งขอบเขตและการเชื่อมโยงเพื่อไม่ให้ลงทุนซ้ำซ้อน"

**มุมความคุ้มค่า:** "โครงการฝึกอบรม __ ตั้งงบ __ ลบ. เป้าหมาย __ คน คิดเป็น __ บาท/คน ขอทราบเกณฑ์ต้นทุนต่อหน่วย และผลการนำความรู้ไปใช้จากการอบรมปีที่ผ่านมา"

**มุมครุภัณฑ์/ราคากลาง:** "รายการ [ครุภัณฑ์] จำนวน __ หน่วย ราคาหน่วยละ __ บาท สูงกว่าบัญชีราคามาตรฐานครุภัณฑ์ที่ __ บาท ขอทราบที่มาราคาและคุณลักษณะที่ต่างจากมาตรฐาน / เสนอพิจารณาปรับลดส่วนต่าง __ ลบ."

**มุมงบผูกพัน:** "โครงการก่อสร้าง __ วงเงินรวมทั้งโครงการ __ ลบ. ตั้งปีนี้ __ ลบ. ก่อภาระผูกพันปี ๒๕๗๑–__ อีก __ ลบ. ขอทราบความพร้อมด้านแบบรูป/พื้นที่/EIA และแผนการเบิกจ่าย เพื่อประเมินความเสี่ยงการเบิกจ่ายไม่ทัน (ม.๔๑ พ.ร.บ.วิธีการงบประมาณฯ)"

# แผนที่เอกสารงบประมาณ + วิธีค้นข้อมูลหน่วยงาน

ใช้หาว่าข้อมูลที่ต้องการอยู่ไฟล์ใด ก่อนดึงหน้ามาวิเคราะห์ (ตัวเลข/ชื่อไฟล์อ้างอิงจากชุดเอกสาร **ปีงบประมาณ พ.ศ. 2570** ในโปรเจกต์นี้; ปีอื่นโครงสร้างคล้ายกัน)

## ภาพรวมเอกสารงบประมาณ ฉบับที่ 1–7
| ฉบับ | ชื่อ | ใช้ตรวจอะไร |
|---|---|---|
| 1 | รายรับ-รายจ่ายเปรียบเทียบ | ภาพรวมประเทศ, เทียบปีก่อน, ขาดดุล, จำแนกงบรายจ่าย |
| 2 | ประมาณการรายรับ | ฐานรายได้/สมมติฐาน (มุมมหภาค) |
| 3 | **งบประมาณรายจ่าย (แยกตามหน่วยงาน) — 19 เล่ม** | **เอกสารหลักในการตรวจรายหน่วยงาน**: แผนงาน/ผลผลิต/โครงการ/งบรายจ่าย/ครุภัณฑ์/สิ่งก่อสร้าง/งบผูกพัน |
| 4 | งบประมาณรายจ่าย (จำแนกเพิ่มเติม/รายงานประกอบ) | ข้อมูลจำแนกตามยุทธศาสตร์/ผลการดำเนินงาน — เปิดยืนยันสารบัญ |
| 5 | รายงานภาวะเศรษฐกิจและการคลัง | บริบทมหภาค |
| 6 | งบประมาณรายจ่าย (ฉบับเสริม) | ข้อมูลจำแนกเพิ่มเติม — เปิดยืนยันสารบัญ |
| 7 | สถานะและแผนใช้จ่าย **เงินนอกงบประมาณ** | ตรวจสภาพคล่อง/เงินนอกงบฯ ที่ควรใช้ก่อน (มุม 1 & 3) |

เอกสารประกอบอื่น: `ร่าง พ.ร.บ.งบประมาณฯ`, `งบประมาณโดยสังเขป` (สรุปทั้งเล่ม อ่านเร็วก่อนเจาะ), `บันทึกวิเคราะห์สรุปสาระสำคัญฯ` (กรอบวงเงิน/นโยบาย), `รายงาน RIA`, `รายงานสรุปผลรับฟังความคิดเห็น`

## วิธีดึงข้อมูล (เทคนิค)
- **อ่านตรง:** ใช้ Read กับไฟล์ PDF ได้เลย (ระบุช่วงหน้าถ้าไฟล์ใหญ่)
- **ค้นข้ามไฟล์/ตาราง:** ใช้ Bash + pdfplumber (ติดตั้งแล้วในแซนด์บ็อกซ์) เช่น
  ```python
  import pdfplumber
  with pdfplumber.open(PATH) as pdf:
      for i,pg in enumerate(pdf.pages):
          t = pg.extract_text() or ''
          if 'คำที่ค้น' in t: print(i+1, t[:500])
  ```
- **ค้นหน่วยงานในเล่ม:** grep ชื่อกรม/สำนัก แล้วไล่โครงสร้าง แผนงาน → ผลผลิต/โครงการ → งบรายจ่าย → รายการครุภัณฑ์/สิ่งก่อสร้าง → รายการผูกพัน
- **โครงสร้างหน้าในเล่ม (ฉบับ 3):** มักเรียง วิสัยทัศน์/พันธกิจ → งบจำแนกตามงบรายจ่าย → แผนงานพื้นฐาน/ยุทธศาสตร์ → ผลผลิต/โครงการ (มีตัวชี้วัด) → รายละเอียดงบรายจ่าย (บุคลากร/ดำเนินงาน/ลงทุน) → บัญชีครุภัณฑ์-ที่ดิน-สิ่งก่อสร้าง → รายการผูกพันข้ามปี

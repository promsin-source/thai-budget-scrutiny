# เช็กลิสต์ตรวจสอบงบประมาณ 4 มุม

ใช้เป็นคู่มือให้ทีมเอเจนต์ตรวจแต่ละด้าน แต่ละข้อควรจบด้วย **คำถามที่ควรถามหน่วยงาน** และ **การอ้างอิงหน้าเอกสาร**

---

## มุมที่ 1 — วินัยการเงินการคลัง / ภาพรวม
*(เอเจนต์: fiscal-discipline-analyst)*

**ตรวจ**
- [ ] วงเงินรวมของหน่วยงานเทียบปีก่อน เพิ่ม/ลดกี่ % เพราะอะไร มีเหตุผลรองรับส่วนที่เพิ่มหรือไม่
- [ ] จำแนกตามงบรายจ่าย (บุคลากร / ดำเนินงาน / ลงทุน / เงินอุดหนุน / รายจ่ายอื่น) สัดส่วนผิดปกติไหม เช่น เงินอุดหนุน/รายจ่ายอื่นก้อนใหญ่ที่ตรวจสอบยาก
- [ ] รายจ่ายลงทุนเป็นการลงทุนจริงหรือแฝงเป็นรายจ่ายประจำ (ภาพประเทศต้อง ≥ 20% ของวงเงิน และ ≥ วงเงินขาดดุล — พ.ร.บ.วินัยการเงินการคลังฯ ม.20; ปี 2570 อยู่ที่ ~20.8% เฉียดเพดาน)
- [ ] งบกลาง/เงินสำรองจ่ายเพื่อกรณีฉุกเฉินหรือจำเป็น ตั้งสมเหตุผลหรือใช้เลี่ยงกลไกงบปกติ
- [ ] มีเงินนอกงบประมาณ/เงินสะสม/รายได้ของหน่วยงานที่ควรสมทบก่อนหรือไม่ (โยงเอกสารฉบับที่ 7)
- [ ] แผนงานผูกกับยุทธศาสตร์ชาติ/แผนแม่บท/นโยบายรัฐบาลจริง หรือติดป้ายไว้เฉย ๆ
- [ ] ก่อภาระงบประจำระยะยาว (อัตรากำลังใหม่ ค่าเช่าระยะยาว ระบบที่ต้อง MA ต่อเนื่อง) ที่กระทบกรอบการคลังไหม

**ธงเตือน:** วงเงินโตสวนผลงาน, ลงทุนหดแต่ประจำพุ่ง, เงินอุดหนุน/รายจ่ายอื่นก้อนโตไร้รายละเอียด, ตั้งงบทั้งที่มีเงินนอกงบประมาณเหลือมาก

---

## มุมที่ 2 — ความซ้ำซ้อน / ทับซ้อน
*(เอเจนต์: duplication-checker)*

**ตรวจ**
- [ ] ซ้ำภายในหน่วยงาน — หลายสำนัก/กองตั้งโครงการวัตถุประสงค์คล้ายกัน (อบรม สัมมนา ดูงาน จ้างที่ปรึกษา ประชาสัมพันธ์ ระบบ IT)
- [ ] ซ้ำข้ามหน่วยงาน — ภารกิจเดียวกันตั้งหลายกระทรวง/กรม (ศูนย์ข้อมูล/บิ๊กดาต้า แอปพลิเคชัน ศูนย์เรียนรู้ ส่งเสริมท่องเที่ยว งานวิจัยหัวข้อเดียวกัน)
- [ ] ทับซ้อนงบบูรณาการ — ควรอยู่ในแผนบูรณาการ (น้ำ / รัฐบาลดิจิทัล / ต่อต้านทุจริต) แต่ตั้งซ้ำในงบหน่วยงาน หรือกลับกัน
- [ ] ซ้ำกับแหล่งเงินอื่น — มีกองทุน/ทุนหมุนเวียน/เงินกู้เฉพาะกิจ/เงินรายได้รองรับอยู่แล้ว
- [ ] ซ้ำกับ อปท./รัฐวิสาหกิจ — ภารกิจที่ท้องถิ่น/รัฐวิสาหกิจดูแลอยู่แล้ว

**เทคนิคค้น:** ทำดัชนีชื่อโครงการของหน่วยงานเป้าหมาย แล้ว Grep คำสำคัญข้ามเอกสารฉบับที่ 3 ทุกเล่ม เช่น `แพลตฟอร์ม|ศูนย์|บิ๊กดาต้า|ฐานข้อมูล|อบรม|สัมมนา|จ้างที่ปรึกษา|ประชาสัมพันธ์|ดูงาน`
**ระวัง:** แยก "ซ้ำจริง" ออกจาก "คนละพื้นที่/คนละกลุ่มเป้าหมายโดยชอบ"

## มุมที่ 3 — ความคุ้มค่า / เหตุผลความจำเป็น
*(เอเจนต์: value-for-money-analyst)*

**ตรวจ**
- [ ] ผลผลิต-ผลลัพธ์เชื่อมกับเงินที่ขอ ตัวชี้วัดชัดและท้าทาย หรือกว้างจนวัดไม่ได้
- [ ] เหตุผลความจำเป็น — เป็นภารกิจหลักตามอำนาจหน้าที่ ทำไมต้องปีนี้ มีทางเลือกถูกกว่าไหม
- [ ] อัตราการเบิกจ่ายปีก่อน — เบิกจ่ายจริงกี่ % มีเงินกันเหลื่อมปีค้างมากไหม แต่ยังขอเพิ่ม (ดูฉบับที่ 4 และฉบับที่ 1)
- [ ] ต้นทุนต่อหน่วย — งบ ÷ เป้าหมาย (บาท/คน, บาท/ครัวเรือน, บาท/กม.) สูงผิดปกติเทียบโครงการคล้ายกัน
- [ ] กลุ่มเสี่ยงคุ้มค่าต่ำ — ศึกษา/วิจัย/จ้างที่ปรึกษา/ประชาสัมพันธ์/ดูงาน ตปท./จัดอีเวนต์ ตรวจการนำไปใช้จริง
- [ ] สภาพคล่องของหน่วยงานเอง (เงินนอกงบประมาณ/เงินสะสม) ที่ควรใช้ก่อน

**ธงเตือน:** ตัวชี้วัดลอย, เบิกจ่ายปีก่อนต่ำแต่ขอเพิ่ม, ต้นทุนต่อหน่วยสูงกว่ามาตรฐาน, โครงการซ้ำเดิมทุกปีโดยไม่มีผลลัพธ์ใหม่

---

## มุมที่ 4 — ครุภัณฑ์ / ราคากลาง & งบผูกพัน
*(เอเจนต์: cost-commitment-auditor)*

**ครุภัณฑ์/สิ่งก่อสร้าง & ราคากลาง**
- [ ] เทียบราคาที่ตั้งกับ: บัญชีราคามาตรฐานครุภัณฑ์ (สงป.), เกณฑ์ราคากลาง+คุณลักษณะครุภัณฑ์คอมพิวเตอร์ (ดีอี), ราคามาตรฐานสิ่งก่อสร้าง
- [ ] รายการนอกบัญชีมาตรฐาน — ขอที่มาราคา (สืบราคา/ใบเสนอราคา 3 ราย)
- [ ] ความสมเหตุผลของจำนวน เทียบภารกิจ/อัตรากำลัง/ของเดิม (รถ คอมพิวเตอร์ ครุภัณฑ์สำนักงานจำนวนมาก)
- [ ] คุณลักษณะเฉพาะที่อาจล็อกยี่ห้อ/รายเดียว
- [ ] ครุภัณฑ์ราคาสูง/นำเข้า/หรูหราเกินภารกิจ

**งบผูกพันข้ามปี (ม.41 พ.ร.บ.วิธีการงบประมาณ 2561)**
- [ ] รายการผูกพันใหม่ — วงเงินรวมทั้งโครงการ vs ที่ตั้งปีนี้ vs ภาระปีถัด ๆ ไป
- [ ] รายการผูกพันเดิม — สอดคล้องความก้าวหน้าจริงไหม มีโครงการล่าช้า/ยังไม่เริ่มแต่ตั้งงบต่อ
- [ ] สัดส่วนภาระผูกพันกินงบลงทุนใหม่จนขาดความยืดหยุ่นไหม
- [ ] ความพร้อมโครงการสิ่งก่อสร้างใหญ่ (แบบ/EIA/ที่ดิน/พื้นที่) เสี่ยงตั้งแล้วเบิกจ่ายไม่ทัน

**ผลลัพธ์เฉพาะมุมนี้:** ระบุ "วงเงินที่เสนอให้พิจารณาปรับลดได้" เมื่อมีฐานชัด (เช่น ส่วนต่างเหนือราคามาตรฐาน) เป็นข้อเสนอต่อที่ประชุม ไม่ใช่คำวินิจฉัย

---

## เกณฑ์ให้ระดับความสำคัญ
- 🔴 **สูง** — วงเงินสูง + ความเสี่ยง/พิรุธชัด หรือมีประเด็นกฎหมาย/วินัยการคลัง → ควรซักในห้องประชุม
- 🟡 **กลาง** — มีข้อสงสัยแต่ข้อมูลไม่พอ → ขอเอกสาร/คำชี้แจงเพิ่ม
- 🟢 **ต่ำ** — ข้อสังเกตเล็กน้อย → บันทึกไว้/ฝากทบทวน
